## 4. 新设备继续开发建议

### 4.1 先做

1. 拉最新代码
2. 确认 JDK 21
3. 安装前端依赖
4. 启动本地 MySQL / Nacos
5. 执行数据库迁移
6. 重启 `order-service`、`report-service`、`config-service`
7. 启动前端

### 4.2 推荐命令

```bash
git pull origin main

cd /path/to/ChengDD/cdd-frontend
corepack pnpm install

cd /path/to/ChengDD
bash scripts/db/migrate.sh

export CDD_ORDER_SERVER_PORT=8085 && bash scripts/local/run_order_service_mysql.sh
export CDD_REPORT_SERVER_PORT=8088 && bash scripts/local/run_report_service_mysql.sh
export CDD_CONFIG_SERVER_PORT=8089 && bash scripts/local/run_config_service_mysql.sh

cd cdd-frontend
corepack pnpm dev
```

## 5. 新设备优先检查点

- `http://127.0.0.1:4173/dashboard`
- `http://127.0.0.1:4173/config`
- `http://127.0.0.1:4173/aftersales`
- `http://127.0.0.1:8085/api/order/after-sales?merchant_id=1001&store_id=1001`

- `8085` 是否是最新 `order-service`
- 本地库是否已执行到 `V22`
