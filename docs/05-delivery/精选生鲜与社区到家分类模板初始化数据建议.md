# 精选生鲜与社区到家分类模板初始化数据建议

## 用途

本文件用于：

- 分类模板初始化数据设计
- 分类页 / 分类模板页展示文案准备
- 设计稿、前端页面、测试数据统一基线

## 设计原则

- 面向生鲜即时零售、小程序商城、社区到家场景
- 参考头部即时零售与社区到家业态的真实商品组织方式
- 兼顾“餐桌场景”和“日常高频补货”
- 支持三级分类结构
- 适合商家初始化后继续扩展

## 页面初始化建议

### 商家端商品分类页

- 默认优先展示“推荐模板”
- 若商家无分类树，页面主状态提示：
  - 当前尚未初始化商品分类，可从平台模板一键生成基础分类结构
- 初始化按钮文案：
  - 使用该模板初始化
- 初始化成功提示：
  - 已根据模板完成分类初始化，后续可继续新增和调整分类

### 平台端分类模板页

- 模板卡片建议显示：
  - 模板名称
  - 模板版本
  - 适用行业
  - 风格标签
  - 适合门店类型
- 推荐标签建议：
  - 推荐模板
  - 新商家常用
  - 社区到家
  - 品质精选

## 测试与演示建议

- 本地演示至少准备两套模板：
  - 品质精选生鲜模板
  - 社区民生到家模板
- 商家默认演示门店可先初始化“品质精选生鲜模板”
- 页面截图和联调示例中，分类树不要只保留一级，应至少展示二级和三级结构
