# 一期 Figma 页面设计提示词

## 1. 使用说明

- 本文件用于把一期已完成的后端能力转换为可视化后台页面设计输入。
- 适用对象：
  - Figma MCP
  - Figma AI
  - 其他可根据提示词生成页面结构或视觉稿的设计工具
- 推荐使用顺序：
  1. 先使用“全局约束”
  2. 再使用“页面导航结构图提示词”
  3. 再使用“低保真线框总提示词”
  4. 最后逐页使用单页面高保真提示词

## 2. 全局约束

```text
为“程哒哒 ChengDD”设计企业级后台页面，高保真，中文界面，桌面端优先并提供移动端重排方案。

统一风格要求：
- 面向商家运营后台 / 平台管理后台
- 视觉关键词：企业级、可信、清晰、轻量电商、低噪音
- 不要紫色系
- 主色建议：深蓝灰 + 橙色强调
- 背景：浅灰底 + 白卡片
- 卡片圆角 12px 左右
- 页面宽度按 1440 设计
- 左侧导航 + 顶部栏 + 主内容区
- 字体要稳重，不要花哨
- 图表和图标简洁，不要过度装饰
- 所有文案使用中文
- 交互状态要完整：默认、悬停、选中、禁用、异常、空状态
- 同时输出桌面版和移动端适配思路
```

