# 程哒哒状态机图

## 1. 文档说明

- 文档用途：描述核心业务对象的状态流转规则，便于产品、研发、测试统一理解系统行为。
- 当前范围：商家入驻、发布任务、商品、订单四类核心对象。
- 使用建议：与业务流程图、角色泳道图配合阅读。

## 2. 商家入驻申请状态机

```mermaid
stateDiagram-v2
    [*] --> draft
    draft --> submitted: 提交申请
    submitted --> pending_process: 平台开始处理
    pending_process --> supplement_required: 要求补资料
    supplement_required --> submitted: 补交后重新提交
    pending_process --> approved: 审核通过
    pending_process --> rejected: 审核拒绝
    rejected --> submitted: 重新整理后再提交
    approved --> opened: 完成接入与一键开通
    opened --> frozen: 平台冻结
    frozen --> opened: 解冻恢复
    opened --> disabled: 停用
```

## 3. 小程序/店铺发布任务状态机

```mermaid
stateDiagram-v2
    [*] --> pending_check
    pending_check --> pending_release: 校验通过
    pending_check --> failed: 校验失败
    failed --> pending_check: 修复后重试
    pending_release --> releasing: 开始发布
    releasing --> success: 发布成功
    releasing --> partial_success: 部分成功
    releasing --> failed: 发布失败
    partial_success --> pending_check: 修复失败项后重试
    success --> rollbacked: 执行回滚
    failed --> rollbacked: 失败后回退到旧版本
```

## 4. 商品状态机

```mermaid
stateDiagram-v2
    [*] --> draft
    draft --> publish_checking: 发起一键发布
    publish_checking --> on_sale: 校验通过并上架
    publish_checking --> publish_failed: 校验失败
    publish_failed --> draft: 修复后保存
    on_sale --> off_sale: 手动下架
    off_sale --> publish_checking: 再次发布
    draft --> deleted: 删除商品
    off_sale --> deleted: 删除商品
```

## 5. 订单状态机

```mermaid
stateDiagram-v2
    [*] --> pending_pay
    pending_pay --> pending_ship: 支付成功
    pending_pay --> cancelled: 超时未支付或主动取消
    pending_ship --> pending_receive: 商家发货/履约完成
    pending_ship --> after_sale: 用户申请售后
    pending_receive --> completed: 用户确认收货
    pending_receive --> after_sale: 用户申请售后
    after_sale --> completed: 售后处理完成且订单完成
    after_sale --> cancelled: 全额退款关闭订单
```

## 6. 售后单状态机

```mermaid
stateDiagram-v2
    [*] --> pending_merchant
    pending_merchant --> agreed: 商家同意
    pending_merchant --> rejected: 商家拒绝
    agreed --> waiting_return: 需要退货
    agreed --> refunding: 仅退款处理
    waiting_return --> refunding: 用户退货完成
    refunding --> completed: 退款完成
    rejected --> closed: 售后关闭
    completed --> [*]
    closed --> [*]
```

## 7. 首页装修配置状态机

```mermaid
stateDiagram-v2
    [*] --> draft
    draft --> published: 发布成功
    published --> draft: 继续编辑新草稿
    published --> rollbacked: 回滚到历史版本
    rollbacked --> published: 回滚版本重新生效
```

## 8. 首页模块状态机

```mermaid
stateDiagram-v2
    [*] --> disabled
    disabled --> enabled: 商家启用
    enabled --> scheduled: 设置时间窗口
    scheduled --> enabled: 到达生效时间
    enabled --> expired: 超过失效时间
    enabled --> disabled: 手动停用
    expired --> disabled: 结束展示
```
